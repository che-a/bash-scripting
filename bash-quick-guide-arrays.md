## Массивы

```sh
#!/usr/bin/env bash

arr=("one" "two" 3 4)
printf "%s\n" ${arr[@]}
```
```sh
one
two
3
4
```

```sh
# Доступ к элементу массива по индексу:
echo ${arr[0]}

# Вывод
# one
```
Вывод количества элементов массива:
```sh
arr=("one" "two" 3 4)
echo ${#arr[@]}
# 4
```
Вывод длинны элемента массива, указанного его индексом:
```sh
arr=("one" "two" 3 4)
echo ${#arr[1]}
# 3
```
